# 워크북
- 위치 기반 스토리텔링 컨텐츠 플랫폼 어플리케이션
- 주어진 위치로 이동해야 감상할 수 있는 스토리텔링 컨텐츠
- 사용자가 이동함에 따라 스토리가 진행되고 배경음악, 오디오, 메시지 등을 통해 몰입 유도
- 위치 정보를 활용한 미션을 통해 다음 이야기로 진행할 수 있는 인터렉션
